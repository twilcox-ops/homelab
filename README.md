# Homelab

Personal homelab environment running Ubuntu Server.
Used for hands-on practice with Linux administration, security hardening, 
automation, and DevOps tooling.

## Areas
- Linux system configuration and hardening
- Security monitoring with Fail2ban and OpenSearch
- Ansible automation playbooks
- Bash and Python scripts

## Status
Ongoing — adding to this as I learn new tools and break things along the way.
